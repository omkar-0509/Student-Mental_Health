# 🧠 Student Mental Health Analysis

This project analyzes student mental health data to identify stress, anxiety, and burnout patterns using Data Science and Machine Learning techniques.

## 📂 Dataset
- **File:** student_mental_health_burnout_1M.csv  
- Features include:
  - Age
  - Gender
  - Study Hours
  - Sleep Hours
  - Academic Pressure
  - Stress Level
  - Anxiety Level
  - Burnout Risk

## 🚀 Technologies Used
- Python  
- NumPy  
- Pandas  
- Matplotlib  
- Seaborn  
- Plotly  
- Scikit-learn  

## 📊 Project Tasks
- Data Cleaning  
- Exploratory Data Analysis  
- Data Visualization  
- Model Building  
- Prediction
